# Event Module

É um mecanismo para
* [] Disparar eventos
* [] Ouvir eventos
* [] Fazer alguma acão quando ouvir o resto
* [] É a base para muito outros módulos como: http, stream, file system, etc...

## O que vamos aprender ?

* [ ] Utilizar os eventos
* [ ] Disparar eventos
* [ ] Ouvir eventos

* [ ] Entender como ele é a base para os outros módulos
